Kelebihan React

  1. Bisa menuliskan kode HTML dalam Javascript
  2. Meningkatkan performa website karena berbasis komponen, jadi tidak reload ulang saat ada perubahan
  3. React JS mudah dipelajari dan dikuasai
  4. React Native untuk Mobile App Development
  5. Banyak digunakan perusahaan startup, jadi jika mahir bisa lebih banyak kesempatan kerja

Kekurangan React

  1. Perkembangan terlalu cepat sehingga developer susah adaptasi
  2. Dokumentasi buruk karena perkembangan terlalu cepat sehingga tidak ada waktu membuat dokumentasi
  3. React hanya untuk membuat layer UI, jadi harus gunakan teknologi lain untuk melengkapi aplikasi web yang dibangun
  4. React menggunakan JSX, yang dianggap beberapa developer susah dipelajari bagi pemula.
